# bitcoin-price
## Run Locally

Clone the project

```bash
  git clone https://github.com/ehsanshavandi/bitcoin-price.git
```

Go to the project directory

```bash
  cd bitcoin-price
```

Install dependencies

```bash
  pip install
```

Start the server

```bash
  python main.py
```
